# CLI Expense Tracker

A lightweight, local-first Command Line Interface (CLI) tool to track and analyze daily expenses. Built as part of a Computer Engineering programming journey.

## 🚀 Features
- **Data Persistence**: Uses a local `expenses.json` file to store data across sessions.
- **Smart ID Management**: Automatically restores the next available ID from the database.
- **Category Analysis**: View total spending and percentage breakdown by category (Food, Transport, etc.).
- **Monthly Tracking**: Specifically view totals for the current month.
- **Robust Validation**: Includes error handling for invalid IDs and non-numeric amounts.
